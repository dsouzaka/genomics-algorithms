# Genomics Algorithms: From Assembly to Probabilistic Models

A progressive series of from-scratch genomics algorithm implementations, 
covering sequence assembly, gene finding, phylogenetics, and hidden Markov 
models. Developed as part of graduate coursework, using simulated/hypothetical 
datasets designed to illustrate core computational biology concepts.

## Modules

1. **[Sequence Assembly](01_sequence_assembly/)** — Greedy overlap-layout 
   read assembler with k-parameter sensitivity analysis
2. **[ORF Finding & Motif Scoring](02_orf_finding_motifs/)** — Multi-frame 
   ORF detection with position weight matrix motif scoring, validated via 
   simulation and probability theory
3. **[Phylogenetics & Molecular Evolution](03_phylogenetics_molecular_evolution/)** — 
   UPGMA clustering and codon-based coding/non-coding classification using 
   log-odds scoring
4. **[Hidden Markov Models](04_hmms/)** — Dinucleotide Markov models for 
   organism classification and a generative 3-state model for ORF structure

## Skills Demonstrated
- Algorithm design (greedy assembly, UPGMA clustering, Markov/HMM reasoning)
- Probabilistic modeling (log-odds scoring, Markov chains, Poisson/binomial approximations)
- Statistical validation via simulation
- Python (type hints, clean function decomposition)

## Note
Datasets are simulated/hypothetical, created for coursework purposes.
